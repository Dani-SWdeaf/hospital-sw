Información - Alta de cita y Consulta de cita
Atención importante!

Las reglas para el alta son:

  • No se pueden agendar cita en un mismo consultorio a la misma hora.
  • No se pueden agendar cita en un mismo doctor a la misma hora
  • No se pueden agendar cita para ningún paciente a la misma hora ni con menos de 2 horas de diferencia para el mismo día. Es decir, sí un paciente tiene 1 cita de 2 a 3pm, solo padría tener otra el día a partir de las 5pm
  • Un mismo doctor no puede tener mas de 8 citas el mismo día.
